# SpringMVCFilmCRUD



## Overview
* Create an application where a user can:
		* Enter a film's ID to view details
		* Search for a film using keyword
		* Add a new film with all properties
		* Delete a film
		* Edit a film
* Apply appropriate error message if:
		* Film is not found
		* Add film fails
		* Delete film fails
		* Edit film fails
